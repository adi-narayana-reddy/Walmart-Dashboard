# Walmart Sales Dashboard (Excel)

## Project Overview
This project presents an interactive Walmart Sales Dashboard created in Microsoft Excel to analyze sales performance across different categories, regions, segments, and shipping modes.

The dashboard helps identify:
- Top-performing product categories
- Regional sales trends
- Profit distribution
- Customer purchasing behavior
- Shipping mode contribution

## Tools & Features Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning
- KPI Metrics

## Dashboard Insights
- Technology category generated the highest sales
- West region showed strong overall performance
- Standard Class shipping contributed the majority of orders
- Phones and Chairs were among the top-selling sub-categories

## Files Included
- Excel Dashboard File (.xlsx)
- Dashboard Screenshot
- Dataset

## Purpose
The goal of this project is to strengthen Excel dashboarding, business analytics, and data visualization skills using real-world retail sales data.
